# 🌿 BreathEase - Universal Health & Wellness Dashboard

![BreathEase Demo](https://via.placeholder.com/800x400.png?text=BreathEase+Universal+Health+Dashboard)

BreathEase is a premium, fully customized health and wellness web application designed to help **Students**, **Teachers**, and **General Users** manage stress, anxiety, and daily focus. It operates entirely as a standalone frontend application powered by the Gemini AI API—no databases or Firebase required!

## ✨ Key Features

### 🎓 Demographic-Specific Zones
We’ve tailored the toolset for specific user needs:
*   **📚 Student Zone:** Exam Anxiety Relief and study aids.
*   **☕ Teacher Lounge:** Burnout Recovery and vocal rest exercises.
*   **🌿 General Wellness:** Daily affirmations and deep sleep planners.

### ⏱️ Interactive Tools
*   **Pomodoro Focus Timer:** A built-in 25-minute focus timer right in the dashboard.
*   **Box Breathing Visualizer:** A beautifully animated 4-4-4 breathing circle to help you ground yourself instantly.
*   **Mood Tracker Emoji Bar:** Log your daily mood with a single click to get tailored advice.
*   **🆘 Panic Relief Button:** A one-click SOS button for immediate grounding techniques during intense anxiety.

### 🎨 Premium UI/UX Aesthetic
*   **Medical Tech Design:** A calming Teal and Slate color palette (`#0d9488`).
*   **Glassmorphism Effects:** Smooth, translucent cards and dynamic drop shadows.
*   **Light/Dark Mode Toggle:** Flawless transitions between sunlit focus and midnight relaxation modes.

## 🚀 Getting Started

To run BreathEase locally on your machine, follow these simple steps.

### Prerequisites
*   [Node.js](https://nodejs.org/) installed on your machine.
*   A Gemini API Key (set within the application).

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Vedant-ghatole/BreathEase.git
    cd BreathEase
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the development server:**
    ```bash
    npm run dev
    ```

4.  **Open your browser:**
    Navigate to `http://localhost:5173/home.html` (or the local URL provided by Vite).
